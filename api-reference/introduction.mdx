## Аутентификация
Basic Стандартная Basic HTTP authentication scheme (RFC 7617), дополнительная проверка подлинности отправителя запроса не выполняется - учетные данные полностью передаются в запросе. Эта схема предназначена для ускорения процесса интеграции.
Для клиента создается пара key + password, которые он передает в заголовке `Authorization`:

```
Authorization: Basic base64(key:password)
```
