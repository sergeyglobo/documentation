Для старта интеграции запросите перечень доступных сетей и активов вместе с минимальными суммами депозита через `GET /gate/deposit/currencies/{payment_profile}`.

<Steps>
  <Step title="Отправьте запрос">
    Укажите платёжный профиль и, при необходимости, `project_id`.

    <RequestExample>
    ```bash gate/deposit/currencies
  curl --location --request GET 'https://finmar.tech/prepare2/api/v3/gate/deposit/currencies/production' \
--header 'Authorization: Basic b3BlcmF0b3JfZGVtbzojdShQcVJAeEQ1N0c='
    ```
    </RequestExample>
  </Step>

  <Step title="Разберите ответ">
