---
title: "Getting List of Networks and Coins"
description: "How to request supported networks, assets, and minimum amounts for crypto deposit"
sidebarTitle: "1. Networks and Coins"
---

To start integration, request a list of available networks and assets along with minimum deposit amounts via `GET /gate/deposit/currencies/{payment_profile}`.

<Steps>
  <Step title="Send request">
    Specify the payment profile and, if necessary, `project_id`.

    <RequestExample>
    ```bash gate/deposit/currencies
curl --location --request GET 'https://sandbox.finmar.tech/prepare2/api/v3/gate/deposit/currencies/sandbox?project_id=preview' \
  --header 'Authorization: Basic BASE64_LOGIN_PASSWORD'
    ```
    </RequestExample>
  </Step>

  <Step title="Parse response">
    The response contains a list of networks and assets with minimum amounts.

    <ResponseExample>
    ```json Success
  {
    "success": true,
    "result": [
        {
            "crypto_network": "TON",
            "crypto_assets": [
                {
                    "asset_name": "TON",
                    "min_amount": "0.000000002"
                },
                {
                    "asset_name": "USDT",
                    "min_amount": "0.000000"
                }
            ]
        },
        // ... (other networks omitted for brevity)
    ],
    "trace_id": "ef26e8037db54488901bc6507755d375"
}
    ```
    </ResponseExample>

    <Tip>
      Minimum amounts can change dynamically — verify values before showing to the user.
    </Tip>
  </Step>
</Steps>

<Check>
  Save `trace_id` for tracing and log the selected `crypto_network`/`asset_name` pair.
</Check>
