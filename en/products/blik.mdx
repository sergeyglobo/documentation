## General Workflow

```mermaid
sequenceDiagram
    autonumber
    actor Client
    participant Project
    participant Finmar API
    Client->>Project: Select payment method
    Project->>Finmar API: Transfer data to create operation
    Finmar API-->>Project: Return payment page URL
    Project->>Client: Redirect to payment page URL
    Finmar API->>Project: Notification<br>about operation completion
    Project->>Project: Process notification
    Project->>Finmar API: Confirm operation processing
```

 <Note>
  Before starting integration, request a username and password for the test environment in the integration chat.
</Note>
